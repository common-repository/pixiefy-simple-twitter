== Description ==

### Pixiefy Twitter Feed by pixiefy

This plugin will create a new twitter feed widget.

Plugin Features


* Activate and play.
* Easy to control. 
* No conflicts.
* Easy customization options.
& many More


== Installation ==

* Download the plugins .zip file.
* Upload to the plugin directory/section through dashboard.
* Activate the plugin.
* Go to widgets section from Appearance.
* Place the newly created custom Pixify twitter feed widget to your preferred area of theme.
* Customize from the given options as you want them to be.

== Frequently asked questions ==

= How do I get the Twitter widget ID*?
Just login to your twitter account > settings > widgets then "Creat new" to creat a twitter widget. You will find the ID after creating the widget.

= Which version of WordPress is best for this plugin? =
This plugin will work with almost any version.

= What will I get from this plugin? =
This plugin will creat a custom twitter feed widget to your widget area. 

= Can I customize it as I want? =
Yes, you can customize it with the options added in the widget.
